Website Customer Review Odoo Apps

These days use of online shopping websites has exceeded by a lot margin. Online shopping has advantage like, user can get every product on its fingertip. But main problem of online shopping is getting conned by someone or getting faulty piece and no support from reseller at all.

This module allows user to write their view on that particular reseller. That might be a good experience or a bitter one. User can write a review and rate it on reseller's profile page.

User just needs to registered with website and should be logged in to website to rate and review any reseller.

This could help reseller improve their services. Also to warn users from some comming resellers.

Enable Users to Rate and Review on Reseller

To visit reseller's page directly from home page, just go to footer of webpage, Under connect with us, Resellers is provided, click on that link and it will redirect to resellers page.


Resellers link will redirect to another page that will show list of resellers.


Resellers overall ratings will be displayed on top of the page.


User can rate and review particular reseller from given form within resellers page.

User cannot submit review until it is logged in to website. If user is not logged in, it will be redirected to login page and after providing credential, it will redirected back to rate and review page.
